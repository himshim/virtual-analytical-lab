# Virtual Analytical Lab 🧪

A **virtual analytical chemistry lab** built to simulate analytical instruments and concepts for
learning, teaching, and experimentation — **without real instruments or reagents**.

This project is **work in progress** and is being built incrementally.

---

## 🌐 Live Demo
https://himshim.github.io/virtual-analytical-lab/

---

## 🎯 Why this exists
- Real instruments are expensive
- Lab time is limited
- Practice and concept clarity matter

So this project explores:
- Simulated analytical instruments
- Interactive learning experiences
- Safe experimentation for students and educators

---

## 🚧 Project Status
**In progress**  
Features, simulations, and UI are actively evolving.

Things may break.  
That’s part of the lab. 😄

---

## 🧠 Intended Use
- Teaching analytical chemistry concepts
- Student self-learning
- Demonstrations without lab risk
- Experimenting with simulation logic
